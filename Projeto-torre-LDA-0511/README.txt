Instruções para utilização do projeto "torre de Hanoi" - Desenvolvido pelo programador Renan dos Santos Azevedo


Para melhor funcionamento é recomendado utilização de uma IDE (VScode ou Netbeans), mas caso haja indisponibilidade de instalação de uma no momento há possibilidade de executar diretamente no navegador.

=============================   INSTRUÇÕES PARA COMPILAÇÃO MANUAL   =============================

1- Extraia a pasta de arquivos "Projeto-torre-LDA-0511.zip" para o caminho desejado (recomenda-se area de trabalho para melhor facilidade de localização).

2- Pressione Windows + R e digite CMD.

3- Execute o Prompt de comando.

4- Utilize o comando: cd Desktop (caso tenha sido salvo a pasta na área de trabalho).

5- Utilize o comando: cd Projeto-torre-LDA-0511 para acessar a pasta do projeto.

6- Utilize o comando:  javac -cp . Torre.java 

7- Com isso um arquivo de nome "Torre.class" deve ser criado.

8- Para executar esse arquivo utilize o comando: java Torre

9- Será requisitado o input "Digite o número de discos: ".

10- Caso o número inserido seja 0, o programa requisitará novamente um número para execução.

11- Após inserir corretamente um número inteiro o programa exibirá os movimentos realizados pelos discos para conclusão do puzzle.

12- Finalizando com o número de movimentos totais.

13- Compilação e execução com CMD concluída. 


=============================   Comandos úteis para movimentação entre pastas no prompt de comando    =============================

dir = Exibe uma lista de arquivos e subdiretórios em um diretório.

cd = Exibe o nome do diretório atual ou faz alterações nele.

cd.. = retorna a pasta anterior.

cd+subpasta = avança para pastas disponíveis no diretório atual.

cd\ = retorna a pasta raiz.


=============================   EXECUTANDO COM NETBEANS   =============================

1- Extraia a pasta de arquivos "Projeto-torre-LDA-0511.zip" para o caminho desejado (recomenda-se area de trabalho para melhor facilidade de localização).

2- Abra a IDE desejada, para exemplificar será utilizado "Netbeans apache 12.1".

3- Após carregamento das funções da IDE, clique na opção "file" ou "arquivos" localizada na barra de navegação (canto superior esquerdo).

4- Selecione a opção "Open project" .

5- Localize o local onde foi extraído a pasta "Projeto-torre-LDA-0511" e selecione o projeto "Torre-de-hanoi".

6- Open project.

7-No explorador localizado no canto esquerdo superior, navegue entre as pastas a seguir:
	Source Package
		<default package>
			Torre.java

8- Clique duas vezes no arquivo "Torre.java" e ele deverá ser exibido pela IDE.

9- Pressione F6 para executar o programa.

10- Após finalização de carregamento, será iniciada a execução do programa.

11- Será requisitado o input "Digite o número de discos: ".

12- Caso o número inserido seja 0, o programa requisitará novamente um número para execução.

13- Após inserir corretamente um número inteiro o programa exibirá os movimentos realizados pelos discos para conclusão do puzzle.

14- Finalizando com o número de movimentos totais e o tempo decorrido para cálculo (divergente conforme capacidade de processamento do sistema utilizado).

15- Abertura e execução finalizadas.

=============================   CASO NÃO HAJA IDE   =============================

Passo a passo para utilização do código sem a necessidade de uma IDE, utilizando apenas internet e um navegador atualizado.

1- Abra o navegador de preferência. (indicado Google Chrome).

2- Acesse o site https://www.jdoodle.com/online-java-compiler/

3- Marque a opção "interactive", de forma que ela fique com a cor escura e o quadrado na direita.

4- Exclua qualquer linha escrita predefinida no site.

5- Abra o arquivo "Torre-para-jdoodle.txt" na pasta extraída anteriormente.

6- Selecione todas as linhas de código utilizando Ctrl + A.

7- Após selecionar, copie o código utilizando Ctrl + C.

8- Cole no navegador, dentro do editor de texto do Jdoodle utilizando Ctrl + V.

9- Após colar, execute o código(botão azul).

10- Será requisitado o input "Digite o número de discos: ".

11- Caso o número inserido seja 0, o programa requisitará novamente um número para execução.

12- Após inserir corretamente um número inteiro o programa exibirá os movimentos realizados pelos discos para conclusão do puzzle.

13- Finalizando com o número de movimentos totais.

14- Abertura e execução finalizadas.
